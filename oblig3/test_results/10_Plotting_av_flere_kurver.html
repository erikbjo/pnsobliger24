Ofte er det veldig praktisk kunne plotte flere funksjoner i en graf for lettere kunne sammenligne dem. For dette er en for-lkke veldig praktisk. Fra tidligere vet vi at matplotlib og numpy kan brukes til plotte kurver slik som vist nedenfor.

In[1]:
import numpy as np
import matplotlib.pyplot as plt


def f(x):
    return x * x / 2


x = np.linspace(0, 3, 501)
y = f(x)

plt.plot(x, y)
Out[1]:
[<matplotlib.lines.Line2D at 0x1054cb130>]

Nr flere kurver skal plottes i samme graf kan dette gjres ved kalle funksjon plt.plot() flere ganger som i koden nedenfor. Nr vi gjr dette er det vanlig inkludere en merkelapp, eller label, som lar oss se forskjellen mellom de to kurvene.

In[2]:
import numpy as np
import matplotlib.pyplot as plt


def f(x):
    return x * x / 2


def h(x):
    return x


x = np.linspace(0, 3, 501)
yf = f(x)
yh = h(x)

plt.plot(x, yf, label="f(x)")
plt.plot(x, yh, label="h(x)")

plt.legend()
Out[2]:
<matplotlib.legend.Legend at 0x105561ae0>

Denne muligheten for vise flere kurver i samme figur kan ogs anvendes til gi en grafisk fremstilling av funksjoner med mer enn n variabel.

Eksempel p funksjon av flere variabler: normalfordelingskurven

Selv om du ikke enda har hatt emnet statistikk, er sjansene store for at du har sett en normalfordelingskurve fr:

Det er ikke ndvendig forst normalfordelingen for lse opppgaven, men kort fortalt s ser vi her sjansen en tilfeldig variabel med normal sannsynlighetsfordeling har for havne innenfor et gitt omrde av grafen. Som grafen illustrerer, er det en parameter $\sigma$ som forteller oss at variabelen har en gitt sannsynlighet for havne innenfor omrdet $0 \pm \sigma$. Normalfordeling er en sannsynlighetsfordeling som er veldig nyttig til modellere bl.a. mleusikkerhet ved mlinger i den virkelige verden.

Funksjonen som beskriver den generelle normalfordelingskurven er oppgitt som flger:

$$f(x, \mu, \sigma) = \frac{1}{\sigma \cdot \sqrt{2\pi}} \cdot e^{-\frac{(x - \mu)^2}{2\sigma^2}}$$

a)

Din oppgave er bruke et plott for vise normalfordelingskurven $f(x, \mu, \sigma)$ i omrdet $-5 \leq x \leq 5$ gitt parametrene i tabellen nedenfor. Plottet skal ha merkelapper som viser verdien av $\sigma$ for de ulike kurvene. Det skal benyttes en for-lkke for plotte de ulike figurene.

Kurve nr. $$\mu$$ $$\sigma$$
1 0 1
2 0 2
3 0 3
4 0 4

PS: Her er et forslag til hvordan figuren kan se ut ved riktig utfrelse.
Nyttig vite: for skrive mattesymboler i matplotlib-figurer kan vi bruke latex math. Esempel: label=f"$\\sigma = {sigma}$" kan brukes p merkelappene i dette plottet. Dette er dog ikke ndvendig for f oppgaven godkjent.

In[3]:
def normalfordeling(x, mu, sigma):
    return 1 / (sigma * np.sqrt(2 * np.pi)) * np.exp(-((x - mu) ** 2) / (2 * sigma ** 2))


x = np.linspace(-5, 5, 501)

import matplotlib.pyplot as plt

for mu, sigma in [(0, 1), (0, 2), (0, 3), (0, 4)]:
    y = normalfordeling(x, mu, sigma)
    plt.plot(x, y, label=f"$\\sigma = {sigma}$")
    plt.legend()

b)

I deloppgave a) endret vi kun p parameteren $\sigma$ i lpet av plottingen, men ikke p parameteren $\mu$ som er kjent som forventingsverdien (med andre ord, gjennomsnittet til den tilfeldige variabelen over tid). Vi skal n lage en figur der vi viser normalfordelingskurven for forskjellige verdier av $\mu$ og $\sigma$:

Kurve nr. $$\mu$$ $$\sigma$$

2 1 0.6
3 -2 0.8
4 0 2

x = np.linspace(-5, 5, 501)

import matplotlib.pyplot as plt

for mu, sigma in [(0, 1), (1, 0.6), (-2, 0.8), (0, 2)]:
    y = normalfordeling(x, mu, sigma)
    plt.plot(x, y, label=f"$\\mu = {mu}, \\sigma = {sigma}$")
    plt.legend()
